# 🏦 PyBank — Virtual Banking App

A virtual banking web app built with Python and Flask.

---

## ✨ Features
- 👤 Create a bank account with a username and PIN
- 🔐 Secure login authentication
- 💰 Deposit and withdraw money
- 📊 View real-time balance
- 📜 Transaction history tracking

---

## 🛠️ Tech Stack
- 🐍 **Python** — core banking logic
- 🌐 **Flask** — web framework
- 🎨 **HTML/CSS** — frontend design

---

## 🚀 How to Run

1. Clone the repository
   git clone https://github.com/Balenciix/pybank-flask.git

2. Install Flask:
   **pip3 install flask**

3. Run the app:
   **python3 app.py**

4. Open your browser and go to
   http://127.0.0.1:5000

## 💡 Why I Built This

I built PyBank as a way to apply what I was learning in my Python programming course and take it a step further. I wanted to go beyond just writing scripts and actually build something I could interact with like a real app.

Banking apps are something everyone uses every day, so I thought it would be a fun and practical project to recreate the core features from scratch — account creation, login authentication, deposits, withdrawals, and transaction history.
